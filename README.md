# vue_vite起手專案

## Version

- **Node.js** v16.14.2
- **yarn** v1.22.18
- **vue** v3.2.25

## Config

- **.env.development** 開發模式
- **.env.production** 生產模式
- **.env.uat** 測試模式

## Setup

```
yarn install
```

### Compiles and minifies for production

```
yarn run serve 開發模式
yarn build 生產模式
yarn build:uat 測試模式
```

### Deploy

執行 compile 之後根目錄下產生 `/dist` 檔案夾
